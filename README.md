# Zomato Data Analysis Project

## Tools Used
- Python
- Pandas
- NumPy

 Key Analysis Performed
This project analyzes Zomato restaurant data to find:
- Top cities by revenue
- Most popular cuisines
- Rating distribution
- Price trends

Project Files
- `zometoproject.ipynb` → Main analysis notebook  
- `data.xlsx` → Dataset used for analysis
